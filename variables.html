Using variables to control the tempo

Creating and using a variable

Instead of manually setting a value for each sleep code>, you can set a tempo variable, allowing you to use code to change the value.

Set your new variable before while True to make sure it doesn't reset it every time:

tempo = 1000

Inside your loop, change the value inside sleep :

sleep(tempo)

The full code will look like this:

tempo = 1000
while True:
    if button_a.was_pressed():
        pin1.write_analog(35)
        sleep(tempo)
        pin1.write_analog(85)
        sleep(tempo)

Adding random

Now you are set up, you can start having fun with your tempo variable.

One thing to try, is getting the micro:bit to choose the tempo for you, randomly.

Add the following line to your code, above while True:

from random import randint

To choose a new tempo every loop, add a new line of code after your if statement:

tempo = randint(100, 5000)

The first number is the minimum number it can choose, and the second is the maximum.


The full code will look like this:

from random import randint
while True:
    if button_a.was_pressed():
        tempo = randint(100, 5000)
        pin1.write_analog(35)
        sleep(tempo)
        pin1.write_analog(85)
        sleep(tempo)
